This is a automatic sidebar creator plugin. Anyone can place that sidebar using short code in anywhere..

== Description ==
This plugins creates sidebar automatically and create shortcode of that sidebar.
So you can use the sidebar anywhere using that shortcode([custom-sidebar name="your sidebar name"]).
you can also use this shortcode into your page template php code.


== Installation ==

1. Go to plugin>add new and upload sidebar-creator.zip
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Then go to Appearance>Add Sidebar and put your sidebar name into sidebar-1 field if you want more sidebar then click on 

"Add more" button and put another sidebar name here.And save the form.
1.If your want to remove the sidebar, click on remove link.And save the form.

== Frequently Asked Questions ==

= How do I display my sidebar? =
 After create sidebar, go to appearance>widget you can see the sidebar. And you can see the shortcode under the sidebar title 

like [custom-sidebar name="your sidebar name"] copy the code and paste the code where you want to show the sidebar.
